A-Team Internship

This is the A-Team Internship, you need a GPA higher than a 2.5 to qualify.

The web application provides three tabs: Home, Grade Input, Qualified Students.

The Home tab provides information about how the internship approval system works.

On the Grade Input tab, enter each students letter grades for the following classes: CSC141, CSC142, CSC240, and CSC241. The system will compute your partial GPA to see if you qualify.

The Qualified Students tab will allow you to view all the qualified students. Students with a GPA above 2.5
